Cyclone Technologies

Cyclone Technologies Inc. is a responsive web application for a factitious technology company that includes a home page, newsletter, services, testimonials, and a contact page.

Skyscraper Lifts

Skyscraper Lifts is an application designed to simulate an elevator system for a skyscraper with 100 floors. It features a home page with a looping video, an about page showcasing the original prompt and constraints, and various simulation pages demonstrating elevator interactions.

Technologies used: Create React App, React, Redux, Material-UI, Jest, React Testing Library.

Web Forms Frontend

Web Form Frontend is a project designed to display and manipulate API data. It allows users to view, manipulate, and persist submitted form data through various features, including user registration and form management. This application is intended to work alongside the companion API project, Web Form API, to enhance functionality and data interaction.

Technologies used: Next.js 15, Fetch, React Table, React Hook Form, MUI, Jest, React Testing Library.
